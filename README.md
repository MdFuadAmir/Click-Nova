# 🚀 Click Nova - Modern Portfolio Web Application

Click Nova is a modern, responsive, and animated personal portfolio web application built to showcase projects, skills, and services of a MERN Stack Developer in a clean and professional way.

It focuses on **performance, smooth UI/UX, and modern frontend experience**.

---

## ✨ Project Purpose

The main purpose of this project is to:

- Showcase developer portfolio in a modern way
- Present real-world projects with details
- Provide smooth user experience with animations
- Allow clients or recruiters to easily contact the developer
- Build a strong personal branding as a web developer

---

## 🎯 Features

- ⚡ Smooth animated UI using Framer Motion
- 🎨 Modern glassmorphism design
- 🌙 Dark / Light mode support
- 📱 Fully responsive for all devices
- 🧭 Dynamic routing for project details
- 📬 Contact form with email sending system (Nodemailer backend)
- 🚀 Fast and optimized React performance
- 🖼️ Project showcase system with live preview links

---

## 🛠️ Technologies Used

### Frontend:
- React.js
- Tailwind CSS
- Framer Motion
- React Router
- React Hook Form
- React Icons
- React Toastify

### Backend:

- Express.js
